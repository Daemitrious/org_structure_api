## \# Organization Structure API

## 

## API для управления организационной структурой компании.

## 

## Проект позволяет создавать подразделения, добавлять сотрудников, получать дерево подразделений, перемещать подразделения и удалять их в разных режимах.

##

## \- FastAPI

## \- SQLAlchemy

## \- PostgreSQL

## \- docker-compose

## 

## \## Возможности

## 

## Проект поддерживает создание подразделений, создание сотрудников внутри подразделений, получение подразделения с сотрудниками и дочерними подразделениями, ограничение глубины дерева через параметр "depth", перемещение подразделений, проверку циклов в дереве и удаление подразделений.

## 

## Удаление доступно в двух режимах:

## 

## "cascade" — удалить подразделение, сотрудников и дочерние подразделения.

## 

## "reassign" — удалить подразделение и перевести сотрудников в другое подразделение.

## 

## \## Примеры запросов

## 

## Создать подразделение:

## 

## json:



## {

## &#x20; "name": "Backend",

## &#x20; "parent\_id": null

## }

## 

## Создать сотрудника:

## 

## {

## &#x20; "full\_name": "Ivan Ivanov",

## &#x20; "position": "Backend Developer",

## &#x20; "hired\_at": "2024-01-15"

## }

## 

## 

## Получить подразделение с деревом:

## 

## GET /departments/1?depth=2\&include\_employees=true

## 

## Удалить подразделение каскадно:

## 

## DELETE /departments/1?mode=cascade

## 

## Удалить подразделение с переводом сотрудников:

## 

## DELETE /departments/1?mode=reassign\&reassign\_to\_department\_id=2





## 

## \## Бизнес-логика

## 

## Название подразделения не может быть пустым.

## 

## Название подразделения должно быть уникальным внутри одного родителя.

## 

## ФИО и должность сотрудника не могут быть пустыми.

## 

## Нельзя создать сотрудника в несуществующем подразделении.

## 

## Нельзя сделать подразделение родителем самого себя.

## 

## Нельзя переместить подразделение внутрь собственного поддерева.

## 

## Максимальная глубина дерева в запросе - "5".

## 

## При удалении в режиме "cascade" связанные сотрудники и дочерние подразделения удаляются каскадно.
